Vector Tile Advantages

  • Layers base = overlay
  • no fixed scale levels
  • increased performance
  • reduce bandwidth
  • reduce storage
  • dynamic rendering
  • speed of processing
  • 3d

Limitations

  • wgs84
  • vector only
  • generalization artifacts (potentially)
  • feature density, size limit
  • 3d, extrusion of polygons but not ideal for contours
